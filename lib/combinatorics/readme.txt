Pairwise testing (all-pairs analysis, попарное тестирование или попарный анализ, анализ всех пар комбинаций) - 
это современная и эффективная методика тестирования, основанная на том предположении, 
что большинство дефектов возникает при взаимодействии не более двух факторов. Тестовые наборы, 
генерируемые при использовании данной методики охватывают все уникальные пары комбинаций факторов, 
что считается достаточным для обнаружения большего числа дефектов.

Входные данные для модуля AllPairs могут быть заданы как список списков из строк, 
значения которых однозначно характеризуют параметры для тестовых наборов. 
[
        ['мужчина', 'женщина'],
        ['до 25', '25-70', 'после 70'],
        ['дети есть', 'детей нет']
        ]


Нужно импортировать библиотеку
from lib.combinatorics import all_pairs2 as ap # https://forworktests.blogspot.com/2013/11/pairwise-testing.html

Вызов фунции для параметризации
print(list(ap.all_pairs2([
        ['мужчина', 'женщина'],
        ['до 25', '25-70', 'после 70'],
        ['дети есть', 'детей нет']
        ])))

На выходе:
[
['мужчина', 'до 25', 'дети есть'], 
['женщина', '25-70', 'дети есть'], 
['женщина', 'после 70', 'детей нет'], 
['мужчина', 'после 70', 'дети есть'], 
['мужчина', '25-70', 'детей нет'], 
['женщина', 'до 25', 'детей нет']
]
